Create a DbC diagram from data source

This task describes how a DbC diagram is created from existing source code managed by Eclipse. The source code is stored in a project. Visual DbC also provides some example projects that can be created via the example wizard.

Execute the following steps to create a DbC diagram from data source:

  1. Select the project, folder or package that is used as source directory for diagram generation.
  2. Open new wizard DbC Diagram from Data Source.
  3. Define diagram file.
  4. Select "Next >".
  5. Define domain model file.
  6. Select "Next >".
  7. Define a data source and his connection settings.
  8. Select "Finish". The diagram is created and opened. It is filled with the content provided by the data source.
