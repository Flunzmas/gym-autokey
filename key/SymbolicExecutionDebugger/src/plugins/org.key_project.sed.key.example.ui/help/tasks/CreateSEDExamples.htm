Create SED Examples

This task describes how to create an example project. This is a normal Java project which is filled with predefined content as starting point for new users of the Symbolic Execution Debugger (SED) with KeY as symbolic execution engine.

The following examples exists:

  • SED Examples

Execute the following steps to create an example project:

  1. Open New Wizard.
  2. Select "Examples/Symbolic Execution Debugger (SED)/SED Examples".
  3. Select "Next >".
  4. Define a project name like "SED Examples".
  5. Select "Finish". The project is now created and filled with the example content. The 'Readme.txt' file explaining the content is opened.
